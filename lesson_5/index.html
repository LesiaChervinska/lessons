Відступи padding

Властивість padding — це внутрішній відступ. Використовується для відступів від кордону блоку всередину блоку (елементу), а не між блоками (елементами). Значення можуть бути лише додатними, тобто більше або рівне нулю.

Значенням "padding: 30px;" з усіх сторін
Значенням "padding: 30px;" зверху та знизу сторін
Значенням "padding: 30px 20px 50px 10px;" відповідно зверху, зправа ,знизу та зліва
Приклад запису лише 1 сторони -"padding-top: 35px;". Так само можливий запис padding-right,padding-left,padding-bottom

Відступи margin

Властивість margin — Зовнішній відступ. Використовується для відступів між блоками (елементами), рекомендується робити відступи зверху вниз (margin-bottom) зліва на право. Значення можуть бути від'ємними! НЕ успадковується.

Приклад від'ємного відступу margin-left: -50px;
Приклад відступу margin-left: 150px;
Схлопування margin-bottom: 30px; п.с переможець
Схлопування margin-top: 10px;

Властивості для встановлення розмірів елементів

Властивість width встановлює чітку ширину для блочних або блочно рядкових елементів,не працює з рядковими.

Властивість max-width яка обмежує максимальну ширину елементів,для блочних або блочно рядкових елементів,не працює з рядковими.

Властивість min-width яка обмежує мінімальну ширину елементів,для блочних або блочно рядкових елементів,не працює з рядковими.

Властивість height встановлює чітку висоту для блочних або блочно рядкових елементів,не працює з рядковими. НЕ МОЖНА встановлювати фіксовану висоту для контентних елементів, а для прикладів можна.

Властивість max-height яка встановлює максимальну висоту елементів,для блочних або блочно рядкових елементів,не працює з рядковими.

Властивість min-height яка встановлює мінімальну висоту елементів,для блочних або блочно рядкових елементів,не працює з рядковими.

Властивості для зміни алгоритмів, обчислень та відображення елементів

Властивість box-sizing: content-box. Розміри елементу обчислюються за стандартним алгоритмом з додаванням відступів та бордерів.Встановлена ширина 500px без врахуванням відступів по 20px та бордерів по 5px.
Властивість box-sizing: border-box. Розміри обчислюються за замовченням, додаються відступи та бордери. Розміри елементів завжди будуть чіткі та передбачені. Встановлена ширина 500px з урахуванням відступів по 20px та бордерів по 5px.
Властивість overflow зі значенням visible вказана за замовчуванням, відображає увесь контент, навіть за межами заданого розмірурозміру розміру .
Властивість overflow зі значенням hidden відображає контент лише в заданому розмірі елементу,все інше буде сховано.
Властивість overflow зі значенням scroll додасть смудки для прокрутки.Властивість overflow зі значенням scroll додасть смудки для прокрутки.
Властивість overflow зі значенням auto додасть смудки для прокрутки тільки у випадку необхідності

Властивість vertical-align керує вертикальним вирівнюванням блочно-рядкових та рядкових елементів по вертикалі (осі - Y) Має значення:top,middle,bottom або значення у пікселях.

За допомогою властивості display можна змінювати поводження елементу та глобально змінити його сутність. Можна використовувати з наступними властивостями:block,inline, inline-block,none.
